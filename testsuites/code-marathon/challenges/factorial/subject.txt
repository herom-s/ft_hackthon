Subject: Factorial

Write a function that returns the factorial of a given number.

Prototype:
    int factorial(int n);

Place your solution in:
    ~/ft_hackthon/workspace/factorial/factorial.c

Parameters:
    n: non-negative integer (0 <= n <= 12)

Returns:
    The factorial of n.
    factorial(n) = n * (n-1) * (n-2) * ... * 1
    factorial(0) = 1

The result fits in a 32-bit signed integer for n <= 12.
Factorial(13) = 6227020800 would overflow int. You do not need
to handle values of n greater than 12.

Examples:
    factorial(0)  -> 1
    factorial(1)  -> 1
    factorial(5)  -> 120
    factorial(10) -> 3628800
    factorial(12) -> 479001600

Allowed functions: none (standard library only)
